# Terê Studio Landing RC60 — versão para salvar/publicar

Landing page separada do executável e do servidor de licenças.

## Mantém

- Visual premium da opção escolhida: azul escuro, champanhe/dourado e tipografia elegante.
- Sem botão Entrar no topo.
- Planos Inicial, Profissional, Avançado e Personalizado.
- Ciclos 1 mês, 6 meses e 1 ano.
- Checkout integrado ao servidor de licenças/Mercado Pago.
- Página de sucesso com licença exibida uma única vez.
- Página de recuperação de licença via suporte.

## Publicação

Subir apenas os arquivos da landing no GitHub Pages/HostGator/Vercel/Netlify.
Não subir executáveis, ZIP pesado, token Mercado Pago ou segredo de servidor dentro deste repositório.
